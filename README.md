

# Usage

`python3 main.py <course> <quarter> <campus>`

Ex: `python3 main.py INFO200 spring2020 seattle`

# Setup

`pyenv local`

`poetry install`

`python3 main.py <course>`

`python3 main.py <course> <quarter> <campus>`
